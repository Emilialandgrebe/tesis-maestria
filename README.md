# Tesis Maestría - Predicción de Rendimiento de Arroz

## Objetivo
Analizar y predecir el rendimiento del arroz en Argentina utilizando modelos de regresión lineal y robusta.

## Estructura del repositorio

- Data/: datasets originales y procesados
- scripts/: scripts de limpieza, análisis y modelado
- Outputs/: resultados, gráficos y modelos entrenados
- marcoteórico/: documentos teóricos

## Reproducibilidad
Los análisis fueron desarrollados en R/Python.
